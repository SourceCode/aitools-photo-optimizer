# @aitools-photo-optimizer/node

Node.js runtime and CLI for Photo Optimizer.

## Features

- CLI for bulk optimization (`scan`, `build`, `clean`).
- `Sharp`-based high-performance image processing.
- Filesystem storage adapter.
- Concurrency queue.

## CLI Commands

```bash
apo build <glob> --out <dir>
apo scan <glob>
apo clean
```
